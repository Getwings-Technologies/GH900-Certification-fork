## Scenario

---

## Prerequisites
- GitHub account
- Web browser
- Basic understanding of text files

---

## Step 1: Understand GitHub Gists
A GitHub Gist is:
- A lightweight way to share code or text
- Not a full repository
- Ideal for small examples, scripts, or notes

Gists can be:
- Public (visible to everyone)
- Secret (only accessible via link)
