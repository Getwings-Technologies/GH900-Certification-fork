## Lab Objectives
By completing this lab, you will be able to:
- Understand what GitHub Wikis are
- Enable a Wiki for a repository

---

## Prerequisites
- GitHub account
- A GitHub repository (any repo with Wiki enabled)
- Web browser

---

## Step 1: Understand GitHub Wikis
GitHub Wikis are:
- Built-in documentation spaces for repositories
- Written in Markdown
- Ideal for guides, FAQs, and long-form documentation

Wikis are **separate from the codebase** but attached to a repository.

---

## Step 2: Enable the Wiki
1. Open your GitHub repository
2. Go to **Settings**
3. Under **Features**, ensure **Wikis** is enabled

If already enabled, continue to the next step.

---

## Step 3: Open the Wiki
1. Click the **Wiki** tab in the repository
2. Click **Create the first page**

GitHub opens a Markdown editor.
