About This Demo

This website demonstrates how to use GitHub Pages to publish static sites directly from a GitHub repository.

Get Started

  1. Push your HTML, CSS, and other static files to your repository.
  2. Enable GitHub Pages in the repository settings.
  3. Visit your published site at https://<username>.github.io/<repo>/
